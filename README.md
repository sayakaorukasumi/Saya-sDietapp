# さやのダイエット記録アプリ

ブラウザだけで動く、自分専用のダイエット記録アプリ。

## 使い方

1. `index.html` をブラウザで開く（ダブルクリックでOK）
2. 体重・運動を記録 → 「保存する」
3. 薫 と 霞 が一言コメントしてくれる

データは **この端末のブラウザ（localStorage）** に保存されます。
別の端末で使う／バックアップしたい時は、画面下の「データを書き出す」「データを読み込む」を使ってね。

## ファイル構成

```
index.html       画面
styles.css       見た目
exercises.js     運動の種類リスト（自由に編集OK）
partners.js      パートナーのセリフ（自由に追加OK）
app.js           メインロジック
images/
  kaoru.png      ← ここに薫のアイコン画像を置く
  kasumi.png     ← ここに霞のアイコン画像を置く
  placeholder-*  画像が無い時の仮アイコン
```

## カスタマイズ

### セリフを追加する

`partners.js` を開いて、各カテゴリの配列にセリフを追加するだけ。

```js
weight_decreased: [
  "わぁ、さやちゃん、体重減っとるやん！えらいえらい",
  "ここに新しいセリフを追加",   // ← こんなふうに
],
```

`{n}` は数字（体重差・運動分数・連続日数など）に置き換わります。

### カテゴリ一覧

| カテゴリ | 出るタイミング |
| --- | --- |
| `weight_decreased` | 体重が減った時（`{n}` = 減った量 kg） |
| `weight_increased` | 体重が増えた時（`{n}` = 増えた量 kg） |
| `weight_same`      | 体重が±0.1kg以内の時 |
| `exercise_done`    | 運動を記録した時（`{n}` = 合計分数） |
| `streak_milestone` | 3,7,14,30,60,100... 日連続達成（`{n}` = 連続日数） |
| `first_record`     | いちばん最初の記録 |
| `default`          | 該当なし／挨拶 |

### 運動の種類を追加する

`exercises.js` の配列に追加。

### アイコン画像を差し替える

`images/kaoru.png` と `images/kasumi.png` を置けば自動で読まれます。
（無ければ仮の SVG アイコンが表示されます）
